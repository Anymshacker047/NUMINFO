# OSINT Telegram Intel Bot 🔍

- Phone Lookup (Abstract API)
- WHOIS Lookup (no API)

## Setup
```bash
git clone https://github.com/Anymhacker047/NUMINFO.git
cd NUMINFO
pip install -r requirements.txt

Goto botfather on telegram to get api
use abstract api for get another api
change .env.example to .env
fill both column with api after =
